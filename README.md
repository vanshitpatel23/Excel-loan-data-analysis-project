Loan Data Analysis Project

 Overview

This project focuses on analyzing loan data to uncover meaningful insights, identify risk patterns, and support data-driven financial decision-making.
The analysis includes data cleaning, preprocessing, exploratory data analysis (EDA), and dashboard visualization.

 Objectives

* Understand loan approval trends
* Identify key factors affecting loan status
* Detect high-risk applicants
* Generate actionable business insights

---
